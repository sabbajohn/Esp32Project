# subghz_legacy

Port espelho fiel do stack SubGhz legado.

Não editar manualmente os protocolos aqui quando a origem estiver em:
- `components/brl/drivers/subghz`

Use:
```bash
./scripts/sync_subghz_legacy.sh
```

Objetivo: preservar paridade funcional completa dos protocolos para uso no adapter do pipeline portátil.
